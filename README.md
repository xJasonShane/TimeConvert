# TimeConvert

TimeConvert 是一款轻量级时间/日期格式转换工具，支持一键转换粘贴的时间/日期为指定格式，简单易用，功能强大。

## 功能特性

- 🔄 **实时转换**：输入内容自动转换，无需手动点击转换按钮
- 📋 **剪贴板支持**：一键粘贴输入，一键复制结果
- 🌍 **多格式支持**：支持多种时间/日期输入格式
- 🎨 **直观界面**：简洁清爽的GUI设计，功能突出
- 📱 **响应式布局**：自适应窗口大小调整
- 🇨🇳 **中文支持**：完美支持中文日期格式
- 📦 **单文件运行**：无需安装，直接运行
- 📌 **预设格式**：提供常用格式选择，方便快捷
- ✍️ **自定义格式**：支持自定义时间/日期格式
- 🔒 **无声操作**：无弹窗干扰，流畅体验

## 支持的输入格式

- **数字时间戳**：10位或13位时间戳（秒或毫秒）
- **英文日期格式**：
  - 2023-04-15 14:30:25
  - 2023/04/15 14:30:25
  - 15-04-2023 14:30:25
  - 15/04/2023 14:30:25
  - 2023-04-15T14:30:25
  - 2023-04-15T14:30:25.123
- **中文日期格式**：
  - 2023年4月15日 14时30分25秒
  - 2023年4月15日
  - 2023年4月15日14时30分25秒
  - 四月十五日 14:30:25

## 支持的输出格式

### 预设格式
- YYYY-MM-DD HH:MM:SS
- YYYY-MM-DD
- HH:MM:SS
- YYYY/MM/DD HH:MM:SS
- YYYY/MM/DD
- MM/DD/YYYY
- DD/MM/YYYY
- YYYY年MM月DD日 HH时MM分SS秒
- YYYY年MM月DD日
- ISO格式
- 星期，DD月YYYY

### 自定义格式
支持Python datetime模块的所有格式化指令，例如：
- %Y：年份（4位）
- %m：月份（2位）
- %d：日期（2位）
- %H：小时（24小时制）
- %M：分钟（2位）
- %S：秒（2位）
- %A：完整星期名
- %B：完整月份名

## 使用说明

### 操作步骤

1. **启动程序**：直接双击 `TimeConvert_v1.0.exe` 或运行 `python timeconvert.py`
2. **输入时间**：
   - 方式一：直接在输入框中输入时间/日期
   - 方式二：点击「粘贴」按钮从剪贴板粘贴
3. **选择格式**：
   - 从预设格式下拉菜单中选择常用格式
   - 或在自定义格式框中输入自定义格式
4. **查看结果**：转换结果会实时显示在输出框中
5. **复制结果**：点击「复制」按钮将结果复制到剪贴板
